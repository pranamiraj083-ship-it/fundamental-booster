# fundamental-booster
Fundamental Booster is a beginner-friendly Python project that collects user information, demonstrates variables, data types, type casting, input/output, arithmetic operations, f-strings, and built-in functions like type() and id(). It strengthens core Python programming concepts through hands-on practice.

## 📌 Features

- Collects user information using `input()`
- Displays formatted output using f-strings
- Demonstrates variables and data types
- Uses type casting (`int()` and `float()`)
- Calculates the approximate birth year
- Displays the data type and memory address of each variable
- Uses the `datetime` module to get the current year
